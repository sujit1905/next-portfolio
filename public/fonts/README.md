# Clash Display Font Files

Download Clash Display from Fontshare (free for personal & commercial use):
https://www.fontshare.com/fonts/clash-display

Download the following WOFF2 files and place them in this directory:
- ClashDisplay-Regular.woff2
- ClashDisplay-Medium.woff2
- ClashDisplay-Semibold.woff2
- ClashDisplay-Bold.woff2

Until these files are present, the site falls back to system-ui sans-serif.
The font is referenced in app/layout.tsx via next/font/local.
